# OpenPit C++ SDK

The OpenPit C++ SDK is a C++17 interface for embedding the OpenPit pre-trade
risk engine in CMake projects.

Install the package generated by `cmake --install` and consume it with:

```cmake
find_package(OpenPit CONFIG REQUIRED)
target_link_libraries(your_target PRIVATE OpenPit::openpit)
```

Use `<openpit/openpit.hpp>` for the complete SDK surface, or include
`<openpit/fwd.hpp>` in precompiled headers when only forward declarations are
needed.

The package resolves the native OpenPit runtime at configure time. Provide a
local runtime with `OPENPIT_RUNTIME_LIBRARY` or `OPENPIT_RUNTIME_DIR`, or let
the resolver download the matching release asset.
